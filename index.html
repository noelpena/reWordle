reWordle

How to Play

Use the date picker to select the WORDLE for the selected date. By default yesterday's date is selected.

reWORDLE only allows you to play previous days. If you want to play today's WORDLE, please play the original game.

Instructions

Guess the WORDLE in six tries.

Each guess must be a valid five-letter word. Hit the enter button to submit.

After each guess, the color of the tiles will change to show how close your guess was to the word.
